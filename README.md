# products-api

## How to use

### Requirements
 - [Java 11]
 - [Maven]

### Running on terminal

Inside the root folder, run the following command:

```
mvn clean install
mvn spring-boot:run
```
### More information about the api

Check the  [Swagger API Documentation], and the [Postman Collection].
